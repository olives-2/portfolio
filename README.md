# portfolio

![Last commit](https://img.shields.io/github/last-commit/olives-2/portfolio/main)

This repo contains my portfolio website, made with SvelteKit and Tailwind CSS.

## Run locally

Install the dependencies:

`bun install`

Then, run the script:

`bun dev`

## Deploy

To deploy the site run:

`bun deploy`

This will use the static SvelteKit adapter and gh-actions to deploy the site.

This *should* also work with `npm`
